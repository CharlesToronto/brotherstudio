## BrotherStudio

Minimal architectural visualization portfolio.

## Development

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3002`.

## Content

- Gallery items: `data/gallery.json`
- Uploaded images: `public/uploads/`
- Placeholder images: `public/gallery/`
- Site name / Instagram / contact: `src/content/site.ts`

## Admin

Open `/admin` to upload/replace/delete images, edit names, and drag & drop to reorder.

## Build

```bash
npm run build
```
